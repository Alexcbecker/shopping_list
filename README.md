# ShoppingList
Simple Android app that implements clean architecture using multi-modules MVVM design.

### Architecture Components
This application implements the following concepts :
- MVVM + Data binding
- Multi-modules
- Navigation Component
- ViewModel
- LiveData
- RxJava
- Koin
- Paging
- Room
